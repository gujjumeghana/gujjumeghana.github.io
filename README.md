# meghanagujju.github.io

Personal portfolio website built with Next.js 14.

## Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework
- CSS custom properties — theming and layout
- Deployed via GitHub Pages

## Features

- Two-column sticky sidebar layout
- Light/dark theme toggle with cursor spotlight effect
- Scroll-spy navigation
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Inspiration

Design inspired by [Brittany Chiang](https://brittanychiang.com).

## Build

```bash
npm run build
npm run start
```
